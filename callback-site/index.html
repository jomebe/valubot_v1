VALORANT 디스코드 어시스턴트 봇

Valubot은 Riot Games의 공식 로그인(Riot Sign-On)으로 사용자가 직접 계정을 연결한 뒤, 디스코드에서 본인의 VALORANT 프로필, 랭크 정보, 매치 기록 요약, 성과 통계, 서버 리더보드를 확인할 수 있는 VALORANT 디스코드 어시스턴트 봇입니다.

디스코드에 Valubot 추가하기 개인정보 처리방침 보기

봇 주요 기능

🔑

Riot 계정 연결

공식 Riot Sign-On(RSO) 연동 방식을 사용하여 사용자의 라이엇 계정을 디스코드 유저 ID와 안전하게 일대일 연결합니다.

👤

VALORANT 프로필 조회

디스코드 서버에서 즉시 본인의 발로란트 게임 내 닉네임, 태그라인 및 계정 레벨 정보를 카드 형식으로 빠르게 확인합니다.

📊

랭크 및 경쟁전 정보 확인

현재 시즌 경쟁전의 티어 등급, RR 점수, 그리고 최근 랭크 변동 내역을 실시간 비주얼 임베드로 제공합니다.

⚔️

매치 기록 요약

최근 경기 매치들의 종합적인 승패 결과, 킬/데스/어시스트(KDA), 최종 게임 스코어 및 MVP 달성 지표를 요약하여 제공합니다.

📈

개인 성과 통계

최근 경기 실적 분석 데이터를 통계화하여 실력의 변동 추이를 체크하고 더 전문적인 성과 피드백 분석을 제공합니다.

🏆

디스코드 서버 리더보드

봇이 초대된 디스코드 서버 내 다른 사용자들의 발로란트 랭킹(리더보드)을 생성하고 함께 경쟁할 수 있습니다.

간편 연동 단계 (How it works)

1

Discord 서버에 Valubot을 추가합니다.

상단의 버튼을 통해 사용 중인 디스코드 채널 서버에 봇을 설치합니다.

2

Discord에서 "/로그인" 명령어를 실행합니다.

연결하고자 하는 디스코드 대화방에서 연동 세션 생성을 위해 로그인 명령어를 전송합니다.

3

Riot 공식 로그인 페이지에서 본인 계정으로 로그인합니다.

보안이 적용된 공식 라이엇 OAuth 인증 포털로 안전하게 이동하여 로그인을 진행합니다.

4

계정 연결 후 Discord에서 Valubot 명령어를 사용합니다.

연동이 정상 승인되면 디스코드로 복귀하여 본인의 다양한 매치 데이터 조회를 시작할 수 있습니다.

5

언제든지 "/로그아웃"으로 연결을 해제할 수 있습니다.

봇에서 본인의 연동 계정 정보를 삭제하고 연동을 끊으려면 명령어 하나로 모든 정보가 영구 삭제됩니다.
