발루봇 개인정보 처리방침

최종 업데이트: 2026-04-11

1. 개요

본 개인정보 처리방침은 발루봇 운영팀(이하 "운영자")이 제공하는 Discord 봇 "발루봇"(이하 "봇")과 관련하여 어떤 정보를 수집하고, 왜 사용하며, 어떻게 보관 및 보호하는지 설명합니다.

2. 수집하는 정보

구분 항목 수집 목적
Discord 식별 정보 사용자 ID, 서버 ID, 서버 설정에 사용된 채널 ID, 사용자 태그 명령어 처리, 서버별 설정 저장, 사용자별 기능 제공
발로란트 연동 정보 계정명, 태그, 리전, PUUID, 티어, 계정 레벨, 카드 ID, 갱신 시각 전적/티어/프로필 기능 제공 및 계정 연동 유지
로그인 세션 정보 (선택 기능) 액세스 토큰, ID 토큰, 엔타이틀먼트 토큰, 쿠키, PUUID, 플레이어명, 리전 로그인 기반 상점/지갑/스킨 조회 기능 제공
기술 정보 오류 상태 코드, 예외 메시지 등 서비스 운영 로그(최소한) 보안, 성능 모니터링, 장애 분석

참고: 명령어 원문 및 일반 메시지 본문은 별도 데이터베이스에 장기 저장하지 않습니다.

3. 정보 이용 목적

  • 봇 기능 제공 및 명령어 응답
  • 사용자 요청 처리 및 계정 연동 유지
  • 서비스 안정성 확보, 오류 수정, 보안 대응
  • 이상 사용 패턴 탐지 및 남용 방지

4. 정보 보관 및 파기

  • 발로란트 연동 정보: 사용자가 ㅂ발로삭제 명령으로 연동 해제할 때 삭제되며, 별도 삭제 요청 시 추가 삭제를 진행합니다.
  • 서버 설정 데이터: 봇 운영에 필요한 동안 보관하며, 서버 관리자 요청 시 삭제를 진행합니다.
  • 로그인 세션 정보: 메모리 기반 임시 저장이며 QR 세션은 최대 5분, 로그인 세션은 최대 55분 후 자동 만료됩니다.
  • 임시 파일: 생성 후 30분이 지나면 자동 정리됩니다.
  • 운영 로그: 서비스 안정성 목적의 최소 범위로 처리하며, 장기 보관은 호스팅 제공자 정책 범위 내에서만 이루어질 수 있습니다.

5. 제3자 제공 및 처리위탁

운영자는 원칙적으로 개인정보를 판매하지 않습니다. 다만 다음과 같은 외부 서비스가 처리에 관여할 수 있습니다.

  • Discord API (플랫폼 상호작용 처리)
  • Google Firebase (Cloud Firestore) (데이터 저장 및 운영)
  • HenrikDev Valorant API (api.henrikdev.xyz) (전적/계정 정보 조회)
  • Riot Games 인증/게임 API 엔드포인트 (로그인 기반 기능 제공 시)

법령에 근거가 있거나 수사기관의 적법한 요청이 있는 경우에 한해 필요한 범위에서 정보를 제공할 수 있습니다.

6. 이용자 권리

이용자는 자신의 정보에 대해 열람, 정정, 삭제를 요청할 수 있습니다. 요청은 아래 문의 채널로 접수할 수 있으며, 운영자는 합리적인 기간 내 처리합니다.

7. 보안 조치

운영자는 접근 권한 통제, 비밀정보 관리, 로그 점검 등 합리적 수준의 기술적/관리적 보호조치를 적용합니다.

8. 아동의 개인정보

봇은 아동을 직접적인 대상으로 하지 않으며, Discord 플랫폼의 정책 및 연령 요건을 따릅니다.

9. 정책 변경

본 방침은 운영 정책 또는 법령 변경에 따라 수정될 수 있으며, 변경 시 본 페이지의 업데이트 날짜를 갱신합니다.
