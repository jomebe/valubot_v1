개인정보 처리방침 (Privacy Policy)

최종 수정일: 2026-06-24

1. 개요

Valubot 서비스 제공자(이하 "제공자")는 사용자의 개인정보 보호를 최우선 가치로 여기며, 관련 법령을 준수합니다. 본 방침은 디스코드 봇 서비스 제공 시 수집하는 정보의 종류, 목적, 보관 및 파기 절차 등에 대해 성실히 안내하고자 작성되었습니다.

2. 수집하는 정보

봇의 안정적인 서비스 이용과 기능 연동을 위해 사용자가 계정 연결 수락 시 다음 정보를 수집할 수 있습니다:

  • 디스코드 사용자 식별자 (Discord User ID): 사용자를 구분하고 계정 매핑 설정을 저장하기 위함
  • 라이엇 계정 고유 식별자 (Riot PUUID 및 Account ID): 발로란트 게임 데이터 조회를 위한 핵심 식별 정보
  • 라이엇 계정 표시명 (Riot Game Name 및 Tagline): 디스코드 내에서 사용자의 프로필로 표시하기 위함
  • 인증 토큰 (Access Token / Refresh Token 또는 세션 데이터): Riot Sign-On(RSO) 연동 시 API 인증 요청을 유지하기 위해 필요
  • 기타 서비스 운영 메타데이터: 봇 사용 명령어 횟수, 오류 상태 분석 등 원활한 모니터링을 위한 최소한의 데이터

3. 저장하지 않는 정보

Valubot은 다음 정보를 어떠한 경우에도 획득, 전송 또는 저장하지 않습니다:

  • 라이엇 계정의 비밀번호 (Riot Password): 사용자는 라이엇 게임즈 공식 보안 인증 창을 통해서만 안전하게 인증을 거칩니다.
  • 디스코드 계정의 비밀번호 (Discord Password)
  • 결제 정보 (신용카드 정보, 은행 계좌 정보 등)

4. 수집 목적

수집된 정보는 다음 목적 이외에는 사용되지 않습니다:

  1. 디스코드 내에서 사용자 요청에 따른 전적 조회 및 프로필 정보 로드
  2. 랭크 등락 추적 및 서버 리더보드 계산을 위한 데이터 비교 연산
  3. 비정상 요청 감지 및 이상 트래픽 유발 차단을 통한 서비스 안정화

5. 제3자 제공 및 위탁

제공자는 사용자의 동의 없이 개인정보를 제3자에게 판매하거나 무단 제공하지 않습니다. 다만, 디스코드 플랫폼 환경 내에서의 동작(Discord, Inc.) 및 발로란트 전적 데이터 요청(Riot Games, Inc.)을 위한 API 통신 과정에서 해당 회사들로 정보가 전송될 수 있습니다.

6. 보관 기간 및 파기

  • 계정 연동 데이터는 서비스 사용을 원치 않아 사용자가 디스코드 내에서 /로그아웃 또는 ㅂ로그아웃 명령어를 실행하여 연동 해제를 요청하면 저장소에서 즉시 영구 삭제됩니다.
  • 수집 목적이 달성되었거나 봇이 서버에서 추방된 경우 관련 데이터는 내부 파기 정책에 따라 안전하게 영구 소멸 조치합니다.

7. 사용자 권리

사용자는 원할 때 언제든지 디스코드 명령어(/로그아웃)를 사용하여 자신의 계정 정보 연결을 삭제하고 파기하도록 권리를 행사할 수 있으며, 이메일을 통하여 상세 내역 조회 및 명시적 영구 삭제를 추가 요청하실 수 있습니다.

8. 보안 조치

제공자는 모든 민감한 세션 정보 및 액세스 토큰 데이터가 저장될 때 안전하게 암호화(AES-256-CBC 등) 조치하고 보관되도록 조치하고 있습니다. 또한 무단 조회를 방지하기 위해 최소한의 보관 서버 접근 제한 정책을 유지하고 있습니다.
